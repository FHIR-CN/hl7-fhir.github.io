HSPC Standard Lab Observation (Profile)

StandardLabObs is the base Model for standard laboratory test results/observations such as hematology, chemistry, coagulation, serology and blood bank. This model will not be used for microbiology, and anatomic pathology data
Profiles:
stdStandardLabObs is the base Model for standard laboratory test results/observations such as hematology, chemistry, coagulation, serology and blood bank. This model will not be used for microbiology, and anatomic pathology data
Extensions:
hspc-ReportingPriorityUrgency Level for result :

the urgency level for which results must be reported to a provider or responsible individual.

hspc-DeltaFlagIndicate the data value in the containing model is a change from a previous instance of the same model :

Takes a value from the valueset listed in the model to indicate the data value in the containing model is a change from a previous instance of the same model.

hspc-VerifiedWhether the information has been verified with the source :

Whether the information in the element that contains this extension has been verified with the source.

hspc-UpdatedWhen this information was last updated :

When this information in the element that contains this extension was last updated.
